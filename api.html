Bluew: A Bluetooth Library.

Bluew a simplified bluetooth library for python, with a clear and easy API, which was heavily influenced by the Requests HTTP library.

class bluew.Connection(mac, *args, **kwargs)[source]¶

A Bluew Connection

Provides a persistent connection with one device.

Basic Usage:

>>> import bluew
>>> mac = 'xx:xx:xx:xx:xx'
>>> device = bluew.Connection(mac)
>>> device.info()
