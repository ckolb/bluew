class bluew.connections.Connection(mac, *args, **kwargs)[source]¶

A Bluew Connection

Provides a persistent connection with one device.

Basic Usage:

>>> import bluew
>>> mac = 'xx:xx:xx:xx:xx'
>>> device = bluew.Connection(mac)
>>> device.info()
